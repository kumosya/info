# 内存管理

## 内存映射

### 虚拟地址空间布局

| 虚拟地址范围 | 描述 | 访问权限 |
| ------------ | ----------- | -------- |
| 0xFFFF F800 0000 0000 - 0xFFFF FFFF FFFF FFFF | 保留 | 不可访问 |
| 0xFFFF F000 0000 0000 - 0xFFFF F800 0000 0000 | 页表区 | 内核态可读写 |

| 0xFFFF A010 0000 0000 - 0xFFFF F000 0000 0000 | 保留 | 不可访问 |
| 0xFFFF A000 0000 0000 - 0xFFFF A010 0000 0000 | 显存映射区 | 内核态可访问 |

| 0xFFFF 8000 6000 0000 - 0xFFFF A000 0000 0000 | 空洞 | 不可访问 |

| 0xFFFF 8000 5000 0000 - 0xFFFF 8000 6000 0000 | 内核堆 | 内核态可读写 |
| 0xFFFF 8000 4000 0000 - 0xFFFF 8000 5000 0000 | 内核栈 | 内核态可读写 |

| 0xFFFF 8000 3000 0000 - 0xFFFF 8000 4000 0000 | 内核.bss段 | 内核态可读写 |
| 0xFFFF 8000 2000 0000 - 0xFFFF 8000 3000 0000 | 内核.data段 | 内核态可读写 |
| 0xFFFF 8000 1000 0000 - 0xFFFF 8000 2000 0000 | 内核.rodata段 | 内核态可读写 |
| 0xFFFF 8000 0000 0000 - 0xFFFF 8000 1000 0000 | 内核.text段 | 内核态可执行 | 

| 0x0000 7FFF FFFF FFFF - 0xFFFF 7FFF FFFF FFFF | 不能使用的区域 | 不可访问 | 

| 0x0000 0000 0000 0000 - 0x0000 7FFF FFFF FFFF | 用户区 | 用户态可读写 |
